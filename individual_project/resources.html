Resources


There are five pages in total: index.html, places.html, eateries.html, shopping.html and resources.html.

I looked at similar websites and their content before coming up with a design for my website. The next step was to select the content of the website; for this I used my personal experience as well as internet resources to shortlist the best places to see, eateries, and shopping centers in Bangalore. I wrote the description for all of these places and downloaded pictures for them using Google image search (references for which are stated below). Finally, I designed a logo for my website using photoshop. Once the content was ready, I started developing the website. During the development, I incorporated Google Maps for each page to display the location of the places. I also wanted to display ratings for the attractions and eateries I had listed. For the attractions, I used widgets from Trip Advisor that display the ratings of each place. For eateries, I used Zomato's API to make my own widget to display the rating; and in addition, I also displayed the cuisine for each eatery using the data collected from the API. Finally, to place the cherry on the cake, I added a weather widget to display the current weather in Bangalore using weatherwidget.io.
